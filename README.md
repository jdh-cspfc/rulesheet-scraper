there exist many different resources on the internet providing pinball rulesheets
this is a script designed to gather and list the links from all those resources.
if changes are detected when scraping a source (added or deleted links), it will archive the outdated list for that source to help track changes.
no actual pages or data are downloaded, only lists of machines and associated links are returned.

new sources can be added on the sources page, the script may require slight tweaking depending on the structure of the new source, but all in all it's relatively robust.

feel free to use it for whatever, must remain open source
